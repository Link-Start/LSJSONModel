# /plan 命令

实现 LSJSONModel 功能的规划命令。

---

## 使用方法

在 Claude Code 中输入：

```
/plan 实现 @LSSnakeCaseKeys 宏的自动转换功能
```

## 代理行为

1. 分析需求，理解要实现的功能
2. 检查相关文件（宏实现、映射系统）
3. 提供实现方案
4. 创建任务列表
